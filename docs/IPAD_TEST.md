# iPad-Test

1. ZIP entpacken.
2. Ordner auf GitHub Pages, Netlify oder einem lokalen Webserver bereitstellen.
3. URL in Safari öffnen.
4. Teilen-Symbol antippen.
5. „Zum Home-Bildschirm“ wählen.

Ein direktes Öffnen von `index.html` reicht nur für einen Layouttest. Offline-Funktion, Konfiguration und Service Worker benötigen einen Webserver.
