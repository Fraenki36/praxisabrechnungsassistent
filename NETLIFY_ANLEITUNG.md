# Alternative: Veröffentlichung mit Netlify

1. Erstelle ein kostenloses Netlify-Konto.
2. Öffne in Netlify **Add new site** → **Deploy manually**.
3. Lade dieses entpackte Projekt oder das ZIP-Paket hoch.
4. Netlify erzeugt automatisch eine Webadresse.
5. Öffne diese Adresse auf dem iPad in Safari.
6. Wähle **Teilen** → **Zum Home-Bildschirm**.

Für die laufende Versionsverwaltung ist GitHub Pages empfehlenswerter.
